Kotlin & C# - a Comparison of Two Modern Languages

24 December 2019

The programming language Kotlin is often compared to Java. This is only logical, as it was conceived as a better Java by its makers JetBrains and its primary (but not only target) is the JVM. However, the developers openly stated that they took inspiration from many languages, one of them being C#. In Fact, C# itself was heavily inspired by Java and has a similar target group as Kotlin.

As I've been working a lot with C# in recent times, I noticed similarities and differences between the two languages and their development. In this post we will go over some noteworthy examples and see how the languages might have influenced each other and take a guess how they might evolve in the future.
